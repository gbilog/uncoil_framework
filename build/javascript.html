Scroll Spy Plugin

The ScrollSpy plugin automatically updates navigation targets as active based on the scroll position. If you scroll down this page you can see a live demo of the ScrollSpy plugin in the sub navigation on the right. Notice that the dropdown sub items are highlighted as well.

ScrollSpy requires the use of position: relative; on the element you're spying on. In this case, we have added position: relative; to the body element.

After adding position: relative; in your CSS, you can call Scroll Spy.

Affix Menu

The affix plugin toggles position: fixed; on the selected navigation element as you scroll up and down the page so the element is visible at all times. The subnavigation on the right side of this page is a live demo of the affix plugin.

To use the affix plugin, you will need to call it.
