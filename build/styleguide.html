Example:

h3: Uncoil Heading

h6: Uncoil Heading

HTML Body

This is Uncoil's main body font style. Sizing is relative to a user's browser settings and is set to { font-size: 1rem } by default with a { line-height: 1.25 }

Example:

Occupy ramps bicycle rights, unicorn chartreuse banh mi migas direct trade biodiesel meggings affogato poke authentic. Organic umami mumblecore, dreamcatcher hot chicken prism vinyl PBR&B small batch iceland. Brooklyn fanny pack pabst brunch kogi, PBR&B kale chips bicycle rights fixie selfies.

Intro Body Text

This is a class that can be used as an introduction line of text as a lead line of a body paragraph. Apply this style to any <p> element with by applying the .intro class as shown by the example.

Example:

Occupy ramps bicycle rights, unicorn chartreuse banh mi migas direct trade biodiesel meggings affogato poke authentic.
