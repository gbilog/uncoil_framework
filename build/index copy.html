Getting Started With Uncoil

Uncoil is a very flexible framework that has been creating using Sass, Flexbox and jQuery. It is designed with a mobile-first approach to help create modern and responsive websites that are quick to create and easy to scale for any user and any device.

Most users will want to use the Production version of the framework as it is precompiled and flattened to minimize HTTP requests and so that you can just start building.

If you are an advanced user, there is a development version of the framework that use the task-runner Gulp in order to compile Sass to CSS, enable a browser-sync workflow with dynamic page refresh, and a build process for creating a production ready website. We will not get into specifics as most users will not use this feature, however it is available to users after they install npm dependencies and run the gulp command in terminal once positioned in the frameowrk directory.

Images

Responsive Images

Adding the .responsive class to <img> tags will make images responsive-friendly within Uncoil. This class applies max-width: 100%, height: auto, and display: block within their parent containers.

To center images, use the .center-block class on <img> tags as opposed to the .text-center class.
