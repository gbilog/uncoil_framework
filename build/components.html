Uncoil Image Overlay

By applying class .overlay or class .overlay-plus to a div element associated with your images, you can create an overlay effect when hovering over them. The .overlay class darkens the image and if you include HTML/text within the div that the class is applied to it will appear as a part of the overlay. If you use the .overlay-plus class, a plus sign will appear and you will want to leave the div empty.
