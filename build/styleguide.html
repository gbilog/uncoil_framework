Typography

HTML Headings

All HTML heading from <h1> to <h6> are available and feature the following text styles.

Example:

h1: Uncoil Heading

h2: Uncoil Heading

h3: Uncoil Heading

h4: Uncoil Heading

h5: Uncoil Heading
h6: Uncoil Heading

HTML Headings with Subtext

Example:

h1: Uncoil Heading with subtext

h2: Uncoil Heading with subtext

h3: Uncoil Heading with subtext

h4: Uncoil Heading with subtext

h5: Uncoil Heading with subtext
h6: Uncoil Heading with subtext

HTML Body

This is Uncoil's main body font style. Sizing is relative to a user's browser settings and is set to { font-size: 1rem } by default with a { line-height: 1.25 }

Intro Body Text

This is a class that can be used as an introduction line of text as a lead line of a body paragraph. Apply this style to any <p> element with by applying the .intro class as shown by the example.
